# Templates by the Makers of Tailwind CSS

Modern website templates, crafted with Tailwind CSS, React, and Next.js.

Visually-stunning, easy to customize site templates built with React and Next.js. The perfect starting point for your next project and the ultimate resource for learning how experts build real websites with Tailwind CSS.

## UI Kit

### Catalyst
Catalyst gives you the perfect foundation for your own component system — production-ready React components with thoughtfully designed APIs designed to be copied into your codebase and fully yours to customize.

Catalyst was thoughtfully crafted with the latest and greatest versions of the tools we love so that they are easy to maintain, and fun to actually work on.

- Tailwind CSS v4.1
- React v19
- Headless UI v2.1
- TypeScript v5.3

Documentation:
[https://catalyst.tailwindui.com/docs](https://catalyst.tailwindui.com/docs)
